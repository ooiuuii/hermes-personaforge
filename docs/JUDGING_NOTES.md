# Judging Notes

## Usefulness

PersonaForge is a concrete business tool for a near-term creator economy product: paid AI persona packs. It handles the hard operational parts around the emotional demo:

- payment proof,
- profitable delivery decision,
- license manifest,
- runtime activation,
- refund/hold path,
- redaction and asset boundary,
- audit evidence.

## Viability

The project builds on a real commerce rail already proven in the previous submission. The new surface is different: the paid product is a persona-pack experience rather than a generic service or order workflow.

The demo can be run without secrets:

```powershell
npm install
npm test
npm run build
```

`npm test` regenerates and verifies the proof pack.

## Presentation

The video should lead with the emotional moment:

1. A user buys a persona pack.
2. Hermes verifies the paid order and unlocks a license.
3. The runtime character reads a page about itself.
4. The character reacts with voice and a shy expression.
5. The proof pack shows that this was a controlled business operation, not just a character toy.
